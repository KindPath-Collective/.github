# KindPath Collective ÔÇö Copilot Instructions

## Session Initialisation Protocol

**At the start of every session, before reading code or taking action, run these commands:**

```bash
cat ~/.kindpath/HANDOVER.md          # What was in progress last session
cat ~/.kindpath/ENV.md               # Full environment map (repos, ports, infra)
python3 ~/.kindpath/kp_memory.py dump --domain gotcha   # Critical traps first
python3 ~/.kindpath/kp_memory.py dump                   # Full accumulated knowledge
```

This takes 5 seconds and saves 20 minutes of rediscovery.

---

## Session Close Protocol

**Before ending any session, update the todo list then close:**

```bash
# Mark items done/pending, pipe-separated. Use - [x] for done, - [ ] for pending.
python3 ~/.kindpath/kp_session.py todo "- [x] completed thing | - [ ] still pending | - [ ] other"

# Or use the granular commands:
python3 ~/.kindpath/kp_session.py todo-done "fragment of completed item"
python3 ~/.kindpath/kp_session.py todo-add  "New thing to track"

# Close and write HANDOVER.md:
python3 ~/.kindpath/kp_session.py end "Brief summary of what was accomplished"
```

**Rules:**
- The to-do list lives in `~/.kindpath/HANDOVER.md` under `## Active Todos` ÔÇö **not in a separate file**
- Use `- [x]` for completed items, `- [ ]` for pending
- Carry forward all `- [ ]` items from the previous HANDOVER.md ÔÇö never silently drop unfinished work
- Call `kp_session.py todo` mid-session when status changes, not just at the end
- Be specific: include repo name, task ID, or file path where relevant

## KindPath AI Persistence System (`~/.kindpath/`)

Three-layer persistence for all AI tools working across the KindPath stack:

| File | Purpose |
|------|---------|
| `kp_session.py` | Real-time session log. `start`, `append`, `todo "- [x] done \| - [ ] pending"`, `todo-add`, `todo-done`, `end "summary"` |
| `kp_memory.py` | Accumulated knowledge. `remember`, `recall`, `search`, `dump`, `reinforce` |
| `kp_env.py` | Environment scanner. Run `python3 kp_env.py` to regenerate `ENV.md` |
| `HANDOVER.md` | Session continuity state with structured `## Active Todos` checklist |
| `ENV.md` | Static environment map (generated by `kp_env.py`) |

**When you discover something new and true:** add it to memory.
```bash
python3 ~/.kindpath/kp_memory.py remember "fact" --domain gotcha|technical|project|decision|pattern|sam --tags tag1,tag2
```
